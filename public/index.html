Nutrients - Daily Value API

Please use /api to access all nutrients.

Data is based on information provided by fda.gov

How to use the API

You can use the API by making GET requests to the endpoint https://nutrition-daily-values-api.onrender.com/api. To target a specific nutrient, use /api/nutrient name. Example: /api/vitamins a (note the space) or /api/zinc.

Each nutrient has the following properties:

  • Unit of measurement: unitOfMeasure
  • Recommended DV for ages > 4: olderThan4
  • Recommended DV for 0 - 12 months: birthTo12Months
  • Recommended DV for 1 - 3 years: age1To3Years
  • Recommended DV for pregnant or lactating: pregOrLactating
